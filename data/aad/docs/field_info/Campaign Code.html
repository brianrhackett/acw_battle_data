Agency Notes:  Code number of the campaign with which the event is associated. Campaigns are subsets of theaters. The majority of the campaigns in the Commission study are as they appear in the Official Records and Guide-Index. A few changes, however, were made. Some campaigns that Irvine had placed in the Lower Seaboard and Gulf Approach were transferred to the Western Theater. In these campaigns, the troops involved were wholly or mostly Main Western Theater commands and were operating basically out of that theater. In addition, the Official Records and Guide-Index did not specifically include all of the battles identified in the Commission study in a campaign; listing these battles instead as singular events. In order to provide a framework in which to evaluate these battles, the Commission added some campaigns, using analysis from secondary sources and basing them on related events. Some campaigns were deleted or renamed during the course of the study. These are not in the Commission's official final inventory.

NARA Notes:  The Commission maintained an independent "Campaign" table in its data base that included four elements: a three-digit campaign code, name of the campaign, date of the campaign, and rank of the campaign. The full print out of this table appears as the code list for the Campaign Code column.
