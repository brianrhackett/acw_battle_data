Schema
======



==========  =======  ==========
battle      string   Battle
spell       integer  Spell
start_date  date     Start Date
end_date    date     End Date
==========  =======  ==========

battle
------

:title: Battle
:type: string
:format: default
:constraints:
    :minLength: 5
    :maxLength: 6
    :pattern: [A-Z]{2}[0-9]{3}[A-Z]?
    

CWSAC battle identifier


       
spell
-----

:title: Spell
:type: integer
:format: default
:constraints:
    :minimum: 1
    :maximum: 2
    

Spell number of the battle. Only one battle had non-contiguous dates, and two spells.

----------

:title: Start Date
:type: date
:format: default

--------
